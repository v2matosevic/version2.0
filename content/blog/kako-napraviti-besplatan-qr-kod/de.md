# Wie Sie einen kostenlosen QR-Code erstellen (und ihn richtig einsetzen)

Diese kleinen schwarz-weißen Quadrate sind mittlerweile überall. Restaurant-Speisekarten. Event-Plakate. Produktverpackungen. Visitenkarten. QR-Codes haben sich von einer technischen Nischenspielerei zu etwas entwickelt, das Ihre Großmutter beim Sonntagsbrunch nutzt.

Die gute Nachricht: Einen zu erstellen ist kostenlos und dauert etwa 90 Sekunden. Die schlechte Nachricht: Die meisten Menschen machen es falsch.

Ändern wir das.

## Was ist ein QR-Code eigentlich?

QR steht für Quick Response (schnelle Antwort). Es ist ein zweidimensionaler Barcode, der Daten speichert. Ihre Smartphone-Kamera liest ihn und führt eine Aktion aus. Öffnet eine Website. Speichert einen Kontakt. Verbindet mit WLAN. Sendet eine E-Mail.

Im Gegensatz zu alten Barcodes, die nur eine Zahlenreihe enthalten, können QR-Codes URLs, einfachen Text, Telefonnummern, GPS-Koordinaten, vCard-Kontaktdaten und vieles mehr speichern. Das quadratische Muster kodiert Informationen horizontal und vertikal, weshalb so viel auf kleinem Raum Platz findet.

Jedes moderne Smartphone kann sie nativ scannen. Keine App nötig. Einfach die Kamera darauf richten und tippen.

## So erstellen Sie einen QR-Code kostenlos

Hier ist der unkomplizierte Ablauf:

**Schritt 1: Wählen Sie einen Generator.** [QRCode Monkey](https://www.qrcode-monkey.com/) ist solide und kostenlos. Kein Konto erforderlich. [TEC-IT](https://barcode.tec-it.com/hr/QRCode) ist eine weitere zuverlässige Option.

**Schritt 2: Wählen Sie den Inhaltstyp.** URL ist am häufigsten, aber Sie können auch QR-Codes für einfachen Text, E-Mail-Adressen, Telefonnummern, WLAN-Zugangsdaten, vCards und SMS-Nachrichten erstellen.

**Schritt 3: Geben Sie Ihre Daten ein.** Fügen Sie Ihre URL ein oder tippen Sie Ihre Informationen. Kontrollieren Sie doppelt. Ein QR-Code, der auf einen defekten Link zeigt, ist schlimmer als gar kein QR-Code.

**Schritt 4: Anpassen (optional).** Die meisten Generatoren lassen Sie Farben ändern, ein Logo hinzufügen und die Form der Punkte anpassen. Achten Sie auf Lesbarkeit. Ein QR-Code, der cool aussieht, aber nicht gescannt werden kann, ist nutzlos.

**Schritt 5: Herunterladen.** PNG funktioniert für die digitale Nutzung. SVG oder EPS ist besser für den Druck, weil sie ohne Qualitätsverlust skalieren.

Das war's. Fünf Schritte, zwei Minuten.

## Statische vs. dynamische QR-Codes

Hier übersehen die meisten Menschen einen wichtigen Unterschied.

**Statische QR-Codes** kodieren die Daten direkt. Die URL ist fest im Muster verankert. Wenn Sie das Ziel ändern müssen, brauchen Sie einen komplett neuen QR-Code. Kostenlose Generatoren erstellen meist statische Codes.

**Dynamische QR-Codes** verweisen auf eine Weiterleitungs-URL. Sie können das Ziel jederzeit ändern, ohne den Code selbst zu ändern. Außerdem verfolgen sie Scans, Standorte und Geräte. Die meisten Dienste für dynamische QR-Codes berechnen eine monatliche Gebühr.

Für einen einfachen Link zu Ihrer Website ist statisch völlig ausreichend. Für Marketingkampagnen, bei denen Sie Analysen und Flexibilität wünschen, lohnt sich die Investition in dynamisch.

## Sieben Fehler, die Menschen mit QR-Codes machen

**1. Nicht testen vor dem Druck.** Drucken Sie 5.000 Flyer mit einem defekten QR-Code, und Sie haben 5.000 teure Untersetzer. Scannen Sie Ihren Code immer mit mehreren Smartphones, bevor Sie etwas in den Druck geben.

**2. Zu klein machen.** Ein QR-Code muss mindestens 2x2 cm groß sein, um zuverlässig gescannt zu werden. Für Plakate oder Werbetafeln, die aus der Entfernung gescannt werden, machen Sie ihn größer. Deutlich größer.

**3. Niedriger Farbkontrast.** QR-Codes funktionieren durch den Kontrast zwischen hell und dunkel. Ein dunkelblauer Code auf dunkelgrünem Hintergrund frustriert jeden, der versucht, ihn zu scannen.

**4. Auf nicht-mobile Seiten verlinken.** Jemand scannt Ihren QR-Code mit dem Smartphone. Wenn der Link eine reine Desktop-Seite mit winzigem Text und ohne [responsives Layout](/blog/vaznost-responzivnog-dizajna-na-mobilnim-uredajima/) öffnet, haben Sie die Person bereits verloren.

**5. Kein Kontext.** Ein zufälliger QR-Code ohne Text, der erklärt, was er tut. Würden Sie das scannen? Niemand sonst auch. Fügen Sie einen kurzen Handlungsaufruf hinzu: "Scannen für Speisekarte," "Scannen zum Kontakt speichern," "Scannen für Wegbeschreibung."

**6. Dort platzieren, wo kein Empfang ist.** Ein QR-Code auf einem U-Bahn-Plakat, der auf eine Website verlinkt, ist nutzlos, wenn es unterirdisch keinen Mobilfunkempfang gibt. Denken Sie darüber nach, wo Ihr Code gescannt wird.

**7. Übermäßiges Anpassen.** Starke Anpassungen können die Lesbarkeit des Codes beeinträchtigen. Wenn Sie ein Logo hinzufügen, halten Sie es klein und zentriert. Wenn Sie Farben ändern, bewahren Sie hohen Kontrast. Funktion schlägt Ästhetik.

## Wo QR-Codes wirklich glänzen

**Visitenkarten.** Das ist der Killer-Anwendungsfall. Ein QR-Code auf einer [digitalen Visitenkarte](/services/) ermöglicht es, Ihre vollständigen Kontaktdaten mit einem Scan zu speichern. Kein Tippen. Keine Fehler. Keine verlorenen Papierkarten.

**Restaurant-Speisekarten.** Die Pandemie hat das zum Standard gemacht, und es ist geblieben. Ein Scan, komplette Speisekarte auf dem Handy. Leicht aktualisierbar, kein Neudruck nötig.

**Produktverpackungen.** Verlinken Sie zu Anleitungen, Garantieregistrierung, Zutatenlisten oder Kundenservice. Nützliche Informationen, ohne das Etikett zu überfrachten.

**Event-Marketing.** Plakate, Flyer und Banner mit QR-Codes, die zu Ticketkauf, Veranstaltungsdetails oder Anmeldeformularen führen.

**WLAN teilen.** Generieren Sie einen QR-Code für Ihr Gäste-WLAN-Passwort. Kleben Sie ihn an die Wand. Kein Buchstabieren mehr von "das Passwort ist kleines x großes Y Zahl 7 Unterstrich..."

Das ist ein Gürtel-und-Hosenträger-Ansatz. Der NFC-Chip übernimmt die meisten persönlichen Interaktionen. Der QR-Code deckt alles andere ab. Sie können Ihren QR-Code sogar während Videoanrufen oder Präsentationen auf dem Bildschirm anzeigen.

## Best Practices zusammengefasst

Halten Sie die Daten minimal. Kürzere URLs erzeugen einfachere, besser scannbare Codes. Verwenden Sie bei Bedarf einen URL-Shortener.

Bewahren Sie hohen Kontrast. Schwarz auf Weiß ist nach wie vor die zuverlässigste Kombination.

Testen Sie auf mindestens drei verschiedenen Smartphones vor dem Druck. Android, iPhone, älteres Gerät.

Fügen Sie immer einen kurzen Text bei, der erklärt, was der QR-Code tut. "Scann mich" ist faul. "Scannen Sie für die vollständige Speisekarte" ist klar.

Drucken Sie mindestens 2x2 cm. Größer ist fast immer besser.

Wählen Sie das richtige Format. SVG für Druck. PNG für digital.
