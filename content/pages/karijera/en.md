# We're building. Want to help?

This isn't a corporate job board. There's no "we're a family" speech. We're a small web development company in Zadar that builds custom websites and apps from scratch. When there's more work than hands, we look for good people.

## What It's Like Here

Small team. Real projects. No templates. Every site is built with React, Next.js, and TypeScript. You'll write code that ships to production, not code that sits in a drawer.

We work Monday through Friday, 08:00 to 16:00. No weekend Slack messages. No "quick thing before bed." We build fast during work hours and then we stop.

Projects are interesting. Clients range from local businesses to European companies. You'll work on 3D animations, booking systems, AI integrations, and things we haven't figured out yet.

## Junior Developer

We're looking for someone early in their career who wants to get good at web development. Not someone who already knows everything. Someone who wants to learn.

**What you'll work with:**
- React and Next.js
- TypeScript
- Tailwind CSS
- Git and real deployment workflows
- Client projects from day one (with guidance)

**What we expect:**
- Curiosity. You Google things. You read docs. You try stuff.
- Basic understanding of HTML, CSS, and JavaScript
- Some projects you've built, even small ones, even messy ones
- Willingness to take feedback and improve

**What you get:**
- Hands-on mentorship from senior developers
- Real project work, not busy work
- A clear path from "I'm learning" to "I'm building"
- Paid position. This is not an unpaid internship.

This is genuine mentorship with real project work. You'll make mistakes. That's the point. We'll help you fix them and understand why.

## No Opening That Fits?

We're always interested in good people. If you write code and build things, we want to hear from you. Send us your portfolio or GitHub profile and tell us what you're into.
