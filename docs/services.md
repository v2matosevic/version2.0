# Services

## Core Services (Actively Offered)

### 1. Custom Websites
Hand-coded websites built from scratch. No WordPress, no templates, no page builders. Modern frameworks, fast load times, responsive on every device. Includes business websites, landing pages, multi-language sites, portfolio sites.

### 2. Web Applications
Full-stack web applications with frontend and backend. Member portals, admin dashboards, SaaS tools, booking systems, client portals with authentication, profiles, content access, journals, event management. Whatever the client needs.

### 3. Webshops / E-Commerce
Online stores with payment integration (Stripe, PayPal, local providers), inventory management, order tracking, product catalogs. Built as custom solutions, not WooCommerce or Shopify.

### 4. AI and LLM Integration
AI-powered features built into products. Chatbots, content generation, workflow automation, data analysis, recommendation engines. Not gimmicks but functional tools that solve real problems.

### 5. Digital Business Cards
Custom digital business cards for professionals and teams. Existing product line being carried forward.

### 6. 360° Virtual Tours
Interactive virtual tours for real estate, hospitality, venues, and businesses. Existing product line being carried forward.

### 7. SEO
Technical SEO, on-page optimization, content strategy, site speed optimization, structured data. Focused on results, not reports.

### 8. Integrations
Connect products to existing tools. CRM systems, Slack, Microsoft Teams, Google Workspace, payment gateways, third-party APIs, webhooks, email services.

### 9. Website Maintenance
Ongoing support, updates, security patches, performance monitoring. Includes legacy WordPress site management and redesign for existing WordPress clients.

## Pricing

No fixed packages. Each project is scoped individually.

**Interactive pricing tool:** A multi-step configurator on the website where visitors select project type, features, page count, integrations, and receive an estimated price range. Designed to qualify leads and set expectations before the first call. Replaces the old fixed package system (see decisions.md — Package System resolved 2026-02-22).

**Alternative path:** Direct contact, schedule a meeting, discuss the project scope.

**Price positioning:** Premium. Typical projects €5,000+, minimum €2,500 for basic landing pages. The interactive pricing tool generates custom estimates (see `interactive-pricing-tool.md`). The site should communicate quality and expertise, not affordability.
